# Baseline Diagnostic

Цель baseline-диагностики: не доказать, что "всё уже понятно", а локализовать реальные пробелы до начала полноценной реализации framework.

## Правила

- Проходить без AI, без Stack Overflow и без подглядывания в готовые решения.
- Разрешено использовать только локальную IDE и собственную память.
- Если ответ не знаешь, фиксируй границу незнания явно, а не заполняй её догадками.
- После прохождения обязательно заполнить `results-template.md`.

## Порядок прохождения

1. `php-basics.md`
2. `code-reading.md`
3. `big-o.md`
4. `results-template.md`

## Что оцениваем

- точность базовых знаний PHP;
- качество чтения чужого кода;
- причинное понимание стоимости операций;
- способность отличать уверенный тон от реального знания;
- склонность к магическому мышлению и vibe-кодингу.
