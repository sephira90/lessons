## 3. Небольшой код

```php
<?php
function normalizeEmail(?string $email): string
{
    if (empty($email)) {
        throw new Exception('Email is not exists');
    }

    if (!filter_var(trim($email), FILTER_VALIDATE_EMAIL)) {
        throw new Exception('Format is not valid');
    }
    
    return mb_strtolower($email);
}
```
