## Блок 2. Прогнозирование поведения

Определи, что выведет код, и объясни почему:

```php
$a = ['x' => 1, 'y' => 2];
$b = $a;
$b['x'] = 10;

echo $a['x'] . PHP_EOL;
echo ($a == $b ? 'loose-equal' : 'loose-diff') . PHP_EOL;
echo ($a === $b ? 'strict-equal' : 'strict-diff') . PHP_EOL;
```

Определи, что выведет код, и объясни почему:

```php
function bump(int $value): void
{
    $value++;
}

$n = 5;
bump($n);
echo $n . PHP_EOL;
```

## Блок 3. Небольшой код

Напиши функцию:

```php
function normalizeEmail(?string $email): string
```

Требования:

- `null` и пустая строка должны приводить к исключению;
- пробелы по краям должны быть удалены;
- email должен быть приведён к lowercase;
- при невалидном формате должно бросаться осмысленное исключение.

После кода кратко объясни:

- какие edge cases ты учёл;
- где здесь может появиться случайная сложность.
