# Artifacts

`artifacts/` хранит результаты работы, диагностики, ответы и проверочные материалы.

- `answers/` - ответы пользователя на lesson packs (учебные модули), blind reconstruction (воспроизведение из памяти) и учебные выводы
- `baseline/` - стартовая диагностика и её результаты
- `complexity/` - журналы и отчёты по сложности
- `execution/` - execution log (журнал выполненных разборов, решений и выводов)
- `failures/` - журнал ошибок, ложных гипотез и уроков
- `reviews/` - ревью, postmortem (разбор после события) и trade-off разборы
