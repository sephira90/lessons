# 2026-03-09 - Float Explorer Pack Added

## Context

После отдельного запроса пользователя на понимание `float` в PHP в проект был добавлен отдельный учебный mini-pack (мини-пакет), а не просто объяснение в чате.

## Key Findings

- Простого текстового ответа недостаточно: тема `float` требует predict-verify-explain (предсказал-проверил-объяснил) формата, иначе пользователь быстро получает иллюзию понимания.
- В текущей среде подтверждены ключевые факты:
  - `0.1 + 0.2` даёт `0.30000000000000004`
  - `(0.1 + 0.2) === 0.3` даёт `false`
  - сравнение через `epsilon` даёт практично полезный результат
- При добавлении новых тестов всплыл инженерный дефект: helper-функция `assertSame()` была объявлена в двух тестовых файлах, что сломало общий тестовый запуск. После переименования конфликт снят.

## Decisions

- Добавить отдельный `float-explorer`, а не перегружать `TypeExplorer`.
- Оформить тему как `Unit 1.1a`, то есть как ближайшее углубление после темы типов.
- Зафиксировать отдельный файл ответов, чтобы `float` изучался не как чтение, а как воспроизводимое упражнение.

## What This Changes

- Phase 1 теперь включает рабочий мост от общей темы типов к реальным ограничениям машинной арифметики.
- В проекте появился отдельный учебный инструмент для темы `float`.
- Тестовый контур немного усилен: новый модуль покрыт тестом, а конфликт helper-функций уже выявлен и снят.
