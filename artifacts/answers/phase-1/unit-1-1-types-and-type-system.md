# Unit 1.1 Answers

## Стандартные кейсы

- `0 == '0'` -> true
- `'' == false` -> true
- `'0' == false` -> true
- `null == false` -> true
- `null == 0` -> true
- `0 === '0'` -> false
- `false === 0` -> false
- `'1' == 1` -> true

## Blind Reconstruction

- Что делает `==`: сравнивает значения после неявного приведения типов, если считает такие типы приводимыми
- Что делает `===`: проверяет идентичность и типа и значения (должны совпасть)
- Что меняет `strict_types`: влияет на проверку аргументов и возвращаемых значений функций в объявлениях скалярных типов
- Чего `strict_types` не меняет: поведение операторов сравнения вроде `==` и `===`
