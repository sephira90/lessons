# Lesson Answers

Здесь хранятся ответы пользователя на lesson packs (учебные модули).

Правила:

- не редактировать файлы с заданиями вместо файлов ответов;
- для каждого учебного модуля держать отдельный файл с предсказаниями, объяснениями и blind reconstruction (воспроизведением из памяти);
- сначала фиксировать исходный ответ, потом уже исправления и выводы;
- пользовательский код для упражнений хранить отдельно от ответов, если это помогает не смешивать мышление и реализацию.
