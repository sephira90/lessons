# lessons

Проект для усиления мышления и системного понимания структур данных, алгоритмов и OOP через практическую разработку собственного PHP-фреймворка.

Ключевой принцип: не копировать готовые решения механически, а разбирать причинность, ограничения, trade-offs (компромиссы) и доводить понимание до рабочего кода.

Дополнительный фокус проекта: развить сильное аналитическое и системное мышление, научиться оценивать хорошие практики по контексту и последствиям, а также разбирать сложность кода на уровне алгоритмов, архитектуры, когнитивной нагрузки и цикломатической сложности.
